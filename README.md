# Formula 1 API Project

## Description
This project is a web application that provides information about the Formula 1 2023 season. Users can ask questions about F1 races, drivers, and teams, and receive answers based on the 2023 season data.

## Features
- Interactive Q&A system for F1 2023 season information
- Contact form for user inquiries
- Responsive design for various device sizes

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Python (FastAPI for backend)
- SQLModel for database interactions

## Usage
- Enter a question about the F1 2023 season in the search bar and click "Ask Away" to get an answer.
- Use the contact form to send inquiries or feedback.

## API Endpoints
- GET `/`: Welcome message
- GET `/ask/{user_question}`: Get answer to a specific question
- POST `/contact`: Submit contact form
